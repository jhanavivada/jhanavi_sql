--ART GALLERY MANAGEMENT SYSTEM--

The Art Gallery Management System is a database project developed using MySQL to manage artworks, artists, exhibitions, collectors, and artwork sales efficiently.
This project helps in storing and organizing gallery information such as artist details, artwork collections, exhibition records, collector purchases, and sale details. The database is designed using relational tables with primary keys and foreign keys to maintain proper relationships between entities.

-->FEATURES:-
- Manage artist information
- Store artwork details and prices
- Maintain collector records
- Track exhibitions and artwork sales
- Generate reports using SQL queries
- Perform operations using joins, aggregate functions, and subqueries

-->TECHNOLOGIES USED:-
- MySQL
- SQL Queries
- ER Diagram Design
- GitHub for project management

-->DATABASE CONCEPTS USED:-
- CREATE, INSERT, SELECT
- WHERE clause and operators
- Aggregate functions
- Joins
- Subqueries
- Primary Keys and Foreign Keys
- ER Diagram Relationships
